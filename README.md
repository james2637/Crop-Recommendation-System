# 🌾 AI-Based Crop Recommendation & Advisory System

## 📌 Overview
This project aims to help marginal farmers make better farming decisions using technology.  
We are solving the problem of crop selection by building a system that recommends the best crop to grow based on environmental and soil conditions.

Farmers often rely on traditional methods or guesswork, which can lead to low yield and financial loss. Our application uses a Machine Learning model to provide accurate, data-driven crop recommendations through a simple mobile app.

---

## 🎯 Problem Statement
Marginal farmers face challenges such as:
- Lack of knowledge about suitable crops  
- Changing climate conditions  
- Poor decision-making in crop selection  

This system helps farmers choose the right crop and improve productivity.

---

## 🚀 Solution
We developed a complete system consisting of:
- 📱 Flutter Mobile App (Frontend)
- ⚙️ Spring Boot Backend (API)
- 🤖 Machine Learning Model

The system takes inputs like soil nutrients, temperature, humidity, and rainfall, and predicts the most suitable crop.

## 📱 Frontend (Flutter)
- User-friendly interface for farmers  
- Input form for environmental data  
- Displays crop recommendations  
- Can be extended with multi-language support  

---

## ⚙️ Backend (Spring Boot)
- REST API for communication  
- Handles requests from frontend  
- Sends data to ML model  
- Returns prediction results  

---


## 🔄 Workflow

1. User enters data in the mobile app  
2. Data is sent to backend API  
3. Backend sends data to ML model  
4. ML model predicts the best crop  
5. Result is returned to backend  
6. Backend sends response to app  
7. Farmer sees recommendation
